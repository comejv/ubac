# Hardware Design Files

This folder contains the hardware design files for the UBAC project. The design files are provided in the following formats:

- **Altium Designer**: A proprietary CAD software developed by Altium Corporation.
- **PDF Schematic**: A PDF file containing the schematic diagram of the board.
- **EasyEDA Pro**: A free proprietary CAD software tightly integrated with JLCPCB and LCSC.

Also available is the **Bill of Materials (BOM)** file, which lists all the components used in the design.
